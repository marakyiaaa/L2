Что выведет программа? Объяснить вывод программы. Объяснить как работают defer’ы и их порядок вызовов.

```go
package main

import (
	"fmt"
)


func test() (x int) {//2
	defer func() {
		x++
	}()
	x = 1
	return
}


func anotherTest() int {//1
	var x int
	defer func() {
		x++
	}()
	x = 1
	return x
}


func main() {
	fmt.Println(test())
	fmt.Println(anotherTest()) 
}
```

Ответ:
```
Output:
2
1

Объяснение:

- В функции test() "x" является возвращаемым значением -> 
"х" доступна во всей функции,в том числе и defer.

Поэтапное выполнение функции:

1) "х" == 1
2) return - функция должна вернуть "х",
НО помним, что "х" - возвращаемое значение ->
"х" возвращается после выполнения defer
3) в отложенной функции х++ -> х == 2 теперь ->
функция возвращает х == 2
OUTPUT: 2


- В функции anotherTest() "x" объявлена локально.

Поэтапное выполнение функции:

1) "х" == 1
2) return - функция должна вернуть "х" (т.e скопированное значение х == 1),
НО сначал после выполянется defer
3)в отложенной функции х++ -> х == 2 теперь,
НО это не влияет на значение "x",
т.к. возвращаемое значение "х" уже скопировано в return до этого
OUTPUT: 1
```
