Что выведет программа? Объяснить вывод программы.

```go
package main

func main() {
	ch := make(chan int)
	go func() {
		for i := 0; i < 10; i++ {
			ch <- i
		}
	}()

	for n := range ch {
		println(n)
	}
}
```

Ответ:
```
1) Создаем небуфферизированный канал
2) в горутине запускаем цикл на 10 итераций и записываем значения в канал
После отправки всех чисел горутина завершает свою работу.

3) по циклу выводи значения из канала 

Выведется значения от 0 до 9 и далее deadlock, так как
канал не закрывается и цикл range ожидает новые данные.

Для исправления нужно после цикла for поставить close(ch)
```
